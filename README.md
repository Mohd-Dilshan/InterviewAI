# InterviewAI – AI Powered Interview Coach

InterviewAI is a voice-based AI interview assistant that helps users prepare
for technical and behavioral interviews.

Users can enter their job role and practice interviews with an AI interviewer.
The AI listens to answers, asks follow-up questions, and provides feedback
after the session.

## Features

• Voice-based interview interaction  
• AI-generated interview questions  
• Real-time speech recognition  
• AI feedback and improvement suggestions  
• Interview performance analysis  

## Tech Stack

Frontend
- React / Next.js
- TailwindCSS

Backend
- FastAPI / Node.js

AI
- OpenAI / HuggingFace LLM

Speech
- Web Speech API
- Text-to-Speech


## How It Works

1. User selects job role
2. AI generates interview questions
3. User answers using voice
4. AI analyzes response
5. AI gives feedback and suggestions
